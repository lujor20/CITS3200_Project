What is JOKITOT

Suspicious Document

Large block of text with less runs

Genuine Document

Multiple runs scattered throughout the document

Tool 1

Examines .docx document for metadata and analyse its constituent XML parts. Each run (single editing session) is colour coded to provide a visualisation into how the document was written.

In a genuine document, the number of words will have a positive correlation to the number of runs present. Large documents will on average have higher number of saves and edits, thus, increasing the number of RSID values recorded
